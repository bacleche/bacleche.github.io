Prérequis

Configuration de l'environnement

Installez les outils de modélisation UML et MySQL avant de commencer le TP1.

Deux familles d'outils à installer avant de commencer : un outil de modélisation UML, et MySQL pour la partie base de données. Tout est gratuit et disponible sur Linux comme sur Windows.

01 Outils de modélisation UML

OutilTypeLinuxWindowsUsage recommandé
draw.io (diagrams.net) Visuel, glisser-déposer AppImage ou navigateur .exe ou navigateur Pour débuter — TP1 à TP2
PlantUML UML en texte (diagram-as-code) VS Code + JRE + Graphviz VS Code + JRE + Graphviz Versionner ses diagrammes dans Git — à partir de TP2
StarUML Suite UML complète .deb / AppImage .exe Optionnel — pour aller plus loin

Installation de draw.io

Aucune installation n'est strictement nécessaire — la version web fonctionne identiquement sur les deux OS.

# Version web (recommandée, zéro installation)
# -> https://app.diagrams.net

# Version desktop Linux (AppImage)
wget https://github.com/jgraph/drawio-desktop/releases/latest/download/drawio-x86_64.AppImage
chmod +x drawio-x86_64.AppImage
./drawio-x86_64.AppImage

# Version desktop Windows
# -> Télécharger le .exe depuis https://github.com/jgraph/drawio-desktop/releases

Installation de PlantUML

# --- Linux (Debian/Ubuntu) ---
sudo apt update
sudo apt install -y default-jre graphviz
# Puis dans VS Code : Extensions -> rechercher "PlantUML" (auteur: jebbs) -> Installer

# --- Windows ---
# 1. Installer Java : https://www.java.com/fr/download/
# 2. Installer Graphviz : https://graphviz.org/download/
# 3. VS Code -> Extensions -> "PlantUML" (jebbs) -> Installer

# Vérification (les deux OS, dans un terminal)
java -version
dot -V
Réflexe éditeur Dans VS Code, avec l'extension installée : Alt+D (Linux/Windows) ouvre l'aperçu live d'un fichier .puml. Le diagramme se redessine à chaque sauvegarde.

02 MySQL

# --- Linux (Debian/Ubuntu) ---
sudo apt update
sudo apt install -y mysql-server mysql-client
sudo systemctl enable --now mysql
sudo mysql_secure_installation

# Client graphique (les deux OS)
# -> MySQL Workbench : https://dev.mysql.com/downloads/workbench/

# --- Windows ---
# -> Télécharger "MySQL Installer for Windows" :
#    https://dev.mysql.com/downloads/installer/
# -> Cocher "MySQL Server" + "MySQL Workbench" pendant l'installation

Vérifier que tout fonctionne

mysql -u root -p
mysql> SELECT VERSION();
mysql> SHOW DATABASES;
